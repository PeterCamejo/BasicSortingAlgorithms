Author : The Alex Camejo

Description:

A showcase of the most simple sorting algorithms in each of the three algorithm "categories" (Insertion, Selection, and Exchange) which are the Linear Insertion Sort, Straight Selection Sort, and Bubble Sort respectively. Also includes a showcase of the "naive quicksort" algorithm.

All unit testing done using the Catch Unit Testing Framework from philsquared. Includes for Catch not in this submission.
